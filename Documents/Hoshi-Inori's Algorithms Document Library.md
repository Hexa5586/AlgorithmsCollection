# Hoshi-Inori 的算法文档库

## 概述

这是个人从2026年1月开始的算法题整理项目，时不时会将碰到的一些算法题通过 ISolution 接口集成到项目中，并可通过 Playground 进行测试。

关于 Playground、ISolution 接口等内部机制以及一些内部辅助库的相关说明位于目录 Internal 中。其他目录均包含了对库内各个算法的说明，按一定的规则进行分类存放。
